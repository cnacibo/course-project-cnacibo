# ADR-003: Политики безопасности для исходящих HTTP-запросов

## Status
Дата: 2025-09-22

Статус: Accepted

## Context
При интеграции с внешними сервисами отсутствуют:
- Таймауты для запросов
- Лимиты повторных попыток
- Ограничения на размер ответов

## Decision
Внедрить политики безопасности для исходящих запросов:
1. Таймаут подключения: 5 секунд
2. Таймаут чтения: 30 секунд
3. Максимум 3 повторные попытки с экспоненциальной задержкой
4. Лимит размера ответа: 50MB

## Consequences
- Повышение отказоустойчивости
- Защита от зависаний внешних вызовов
- Контроль использования ресурсов
